# Dynamic-Duo
FESTIKA JAWA TIMUR 2025-Bidang Lomba AREK AI-QUIZ INFORMATIKA INTERAKTIF

#CARA MENJALANKAN PROGRAM
1.	Pastikan Python 3 sudah terpasang/install di computer, jika belum bisa download di https://www.python.org
2.	Klik 2 kali icon Python 3
3.	Setelah dibuka pilihlah lembaran baru atau new file, jika sudah di buat maka bukalah open file
4.	Lalu kalau sudah di buat dan di klik open carilah dokumennya dan masukkan dokumennya
5.	Setelah di masukkan (open) klik bagian Run, lalu pilihlah Run Mobile
6.	Setelah di klik Run Mobile F5 maka program akan berjalan dan siap di kerjakan
7.	Setelah di kerjakan maka nilai hasil akhirnya akan keluar
8.	Program selesai
